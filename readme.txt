Adds a field to Gravity Forms for ranking choices

== Installation ==

You can download and install Gravity Forms Ranking Field using the built in WordPress plugin installer. If you download Gravity Forms Ranking Field manually, make sure it is uploaded to "/wp-content/plugins/gravityforms-ranking-field/".

Activate Gravity Forms Ranking Field in the "Plugins" admin panel using the "Activate" link. If you're using WordPress Multisite, you can choose to activate Gravity Forms Ranking Field network wide for full integration with all of your sites.
